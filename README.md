# CELLINT-RC v2.1
## Cell Intelligence Reconnaissance Console

### Overview
Advanced mobile device analysis tool for Termux that performs:
- Real-time cell tower monitoring
- GPS location tracking
- Device fingerprinting
- IMSI/IMEI analysis
- Network reconnaissance

### Key Features
```plaintext
1. Live cellular monitoring with heatmap visualization
2. Multi-source location tracking (GPS/Network/Fused)
3. Network Cell Info Lite report parsing
4. Device database with historical tracking
5. IMSI/IMEI calculation algorithms
6. WiFi network analysis
7. Automated logging system
